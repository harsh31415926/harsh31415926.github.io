## 🚀 How to Use

### 1. **Local Development**
   - Download all files to a folder
   - Open `index.html` in your web browser
   - No server required!

### 2. **Add Photos**
   - Replace photo placeholders with your actual images
   - Look for `<div class="photo-placeholder">` sections
   - Add image files and update the src path
   
   Example:
   ```html
   <!-- Replace this -->
   <div class="photo-placeholder">
       <p>Add your favorite photo</p>
       <div class="photo-box">📷</div>
   </div>
   
   <!-- With this -->
   <img src="path-to-your-photo.jpg" alt="Photo description" style="width: 100%; height: 100%; object-fit: cover;">
   ```

### 3. **Customize Text**
   - Edit sister's name in the pages
   - Update all the messages and wishes
   - Add your personal stories and memories
   - Customize dates and milestones in the timeline

### 4. **Deploy Online**
   - Upload to GitHub Pages (free)
   - Upload to any web hosting service
   - Share the link with your sister!

## 📝 Customization Tips

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #ff6b9d;      /* Main pink color */
    --secondary-color: #c44569;    /* Darker rose */
    --accent-color: #ffd6e5;       /* Light pink */
}
```

### Add More Pages
1. Create a new `.html` file (e.g., `special.html`)
2. Copy the structure from any existing page
3. Update the navigation links in all pages
4. Add your content

### Customize Messages
- Edit all text directly in the HTML files
- Add your personal stories to the Memory cards
- Update the Letters section with your own message
